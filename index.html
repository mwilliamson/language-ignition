Language ignition

Whenever I start using a new language or platform, I find myself asking the same questions. For instance:

  • How do I install a package, local to my project (rather than system-wide)?
  • How do I install an application written for that platform?
  • How do I set up a new project?

The answer to these questions already exists, but tends to be scattered across different places on the Internet. Language Ignition aims to be a quick reference for this information, with links to existing resources for more detailed information.

This is currently a minimal example of what I had in mind. If you think it might be useful, feel free to get in touch or contribute on GitHub. If you know of an existing site, let me know!
